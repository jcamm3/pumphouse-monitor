# Pumphouse Monitor

ESPHome project running on ESP32-C3 with:
- Dual analog 100 PSI pressure transducers
- SHT30 temp/humidity sensor
- OLED display with cycle pages
- Neopixel RGB LED for state
- Web UI calibration interface
